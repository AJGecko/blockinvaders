# Block Invaders

Block Invaders is a small game were you fight agains evil blocks.

This game was created with microStudio and microScript 2.0.

Try out: https://microstudio.io/AJGecko/blockinvaders/

# Controls

| Action | Input | Description |
|:---|:---|:---|
| Move Left | Left Arrow | Moves the player to the left (only when shield is inactive). |
| Move Right | Right Arrow | Moves the player to the right (only when shield is inactive). |
| Shoot Arrow | Left Click | Shoots an arrow upwards. |
| Activate Shield | Right Click / Spacebar | Activates the shield to block incoming enemy arrows. |
| Menu Navigation | Left Click | Clicks the restart button on the Game Over screen. |
